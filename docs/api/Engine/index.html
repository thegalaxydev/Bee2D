Run​

This is a yielding function. When called, it will pause the Lua thread that called the function until a result is ready to be returned, without interrupting other scripts. Yields
</>
Engine.Run(params: {[string]: any} | nil) → ()

Runs the engine. Goes through all params and sets them if they exist. This initializes the engine and starts the render loop, yielding anything below it indefinitely.
